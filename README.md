# EMUFireNet-Dataset
This repository houses the custom EMU Forest Fire dataset used for testing robustness and accuracy of forest fire detection models. The dataset has two classes: Fire and NoFire.
There are 1830 images under the Fire and 1776 under the NoFire classes. 

The repository also provides the model weights for EMUFireNetv1 and EMUFireNetv2 while using train2 data. Train2 dataset which is used for training both models also has two classes 
and contains 10,800 Fire and 11,144 NoFire images
